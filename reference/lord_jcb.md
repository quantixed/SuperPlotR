# Example cell migration data

Data used in Lord et al 2020 to generate the plots in Figs S4 and S5

## Usage

``` r
lord_jcb
```

## Format

### `lord_jcb`

A data frame with 300 rows and 3 columns:

- Replicate:

  Experimental replicate number (integer)

- Treatment:

  Conditions

- Speed:

  Measurement
