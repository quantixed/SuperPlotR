# FlatPlot Colour Selection (Single Color)

FlatPlot Colour Selection (Single Color)

## Usage

``` r
get_fp_colour(key)
```

## Arguments

- key:

  character name, can be "rl_green", "rl_red", "rl_blue", "rl_yellow",
  "rl_purple", "rl_orange", "rl_magenta", or a hex colour

## Value

character of hex colour

## Examples

``` r
get_fp_colour("rl_green")
#> [1] "#00a651"
```
